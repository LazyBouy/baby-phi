## Plan vs. reality
- **Plan said** (plan-archive line 1201, ¬ßPart 4 P3 Deliverables): `AgentCreated { agent_id, owning_org, kind, role, at, audit_event_id }` ‚Äî field name `kind: AgentKind`.
- **Reality (shipped state at current HEAD)**: `DomainEvent` carries `#[serde(tag = "kind", rename_all = "snake_case")]` at the enum level. Serde's `tag = "kind"` reserves the JSON key `kind` for the variant discriminator; a variant field *also* named `kind` produces the compile-time error "variant field name 'kind' conflicts with internal tag". Shipped as: `AgentCreated { agent_id, owning_org, agent_kind: AgentKind, role: Option<AgentRole>, at, event_id }` ([`events/mod.rs:162-165`](../../../../../../modules/crates/domain/src/events/mod.rs#L162-L165)).
